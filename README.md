# アニメ配信カレンダー

## 概要
昨今はテレビアニメが地上波だけでなく、サブスクリプションサービスで放送直後からチェックできるようになりました。  
自分はテレビでアニメを見なくなってしまい、専らスマートフォンやPCからアニメやバラエティをチェックするようになったのですが、たくさんあるサブスクリプションサービスの中で最速配信はどのアニメなのかを覚えるのがかなり面倒です。

そこで、クール毎にチェックするアニメの配信サブスクリプションサービス・更新曜日・更新時刻を登録することで、自分なりのアニメ視聴用カレンダーを作りました。

## 番組データについて
アイデアは特にないので、番組データは自分でJSONに登録する仕組みにしています。  
番組データ登録は誰でも行えてしまうので、セキュリティという概念は存在しません。将来的にはカレンダーの閲覧は他の人もできるようにしたいです。

また、番組データ登録を自動化できるとなお良いのですが、方法はまったく思いついていません。

## 開発について
このアニメ配信カレンダーは OpenAI Codex で作成しています。
